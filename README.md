# Arcadia Client

🖥️ Standalone game client built on Arcadia Core.

Includes:
- Tile-based world rendering
- Camera system
- UI framework (menus, inventory, dialog)
- Interaction logic (hover, click, drag)

### Future Goals
- Multi-platform support (desktop / web)
- Mod integration
- In-game developer tools
